# Build a Bill Splitter
Practice working with numbers and mathematical operations to build a bill splitter.  This tool will calculate how much each person owes after adding meal costs and a tip.

## Technologies

Uses Python3 and the `+` operator.

## Deployment

Copy `main.py` into the text editor of your choice.  If using terminal, use the command `python3 main.py` to run the code.
